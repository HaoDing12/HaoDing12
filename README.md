## Selected Projects

### [SecureGate - Cloud Security Scanning Platform](https://github.com/HaoDing12/cs6620_final)

AWS-based SAST pipeline that scans pull-request JavaScript changes, posts GitHub Actions results, blocks merges on high-severity findings, and stores scan metadata and reports in DynamoDB and S3. My project scope included the Node.js scanner backend, dashboard API, static dashboard, repo isolation, scanner/dashboard linking, and vulnerability alert path.

Technologies: AWS, Terraform, GitHub Actions, Lambda, EC2, ALB, DynamoDB, S3, SNS, CloudWatch, Docker, Node.js.

### [Rally - Activity-First Social Platform](https://github.com/HaoDing12/rally)

Full-stack social activity platform with activity posting, join requests, compatibility scoring, profiles, ratings, notifications, and automated checks. The repository demonstrates end-to-end product engineering with typed APIs, database modeling, unit tests, E2E tests, and CI.

Technologies: Next.js, TypeScript, React, Prisma, PostgreSQL, NextAuth, Vitest, Playwright, GitHub Actions.
